How to use AlleBot.

 

1. Download ChromeDriver (save it in the AlleBot folder) from: https://chromedriver.chromium.org/

2. In the "Find item" tab provide a name of item you wish to look for in the Allegro.pl website. Use filters to narrow amount of items (speeds up a search time).

3. When search is done excel file with search results will be created in AlleBot folder.
